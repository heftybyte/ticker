# Ticker

Historical &amp; Real-time Token Price Data

RFC: https://docs.google.com/document/d/1pf6wQ2GMgntiYjSX-jK0R3xBpK-h1Xvo8-Fr3L2eUO4/edit#

## Usage

### Get Realtime Prices

```
npm run realtime
```

### Store Historical Prices

```
npm run historical
```

## Dependencies

### Influx DB

OSX

```
brew install influxdb
```
